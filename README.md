# hello-world
Well-written C code to greet the world!

## Example Use

```console
foo@bar:~$./hello_world
Hello, world!
```
## Command Line Options

* -w, --width
   * Terminal Width in Columns for Word Wrap
   *  0 for autodetect

* -n, --nowrap
  -   Disable Word Wrap

* -l, --lang
    - Language

* -h?, --help
   -  Get Help Message

* -v, --version
    - Version Information

* -o, --debugoutput
    - Debug output:
    - 0 for standard output
    - otherwise a filename

* -d, --debuglevel
    - Debug Level:
    - 0 Error
    - 1 Warning
    - 2 Verbose

* -h, --help
     - Display this message.
